# Spotify Clone 
## Created using:
* **React.js**
* **Next.js**
* **NextAuth**
* **Recoil**
* **Tailwind CSS**
* **Spotify API**

## How to use
```
git clone https://github.com/alexwahlgren/spotify-clone.git
npm i
npm run dev
```
Navigate to `http://localhost:3000` to see the app running live on your machine.
